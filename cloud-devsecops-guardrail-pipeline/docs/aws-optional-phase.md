# Optional AWS Phase

This project does not require AWS for Phase 1.

Only do this later.

## Step 1 — Billing Safety

Before creating anything:

- create budget alert
- use free tier only
- do not run Terraform apply
- delete unused resources

## Step 2 — Explore AWS Concepts

Good beginner areas:

- IAM Access Analyzer
- S3 Block Public Access
- CloudTrail basics
- billing alarms
- least privilege examples

## Step 3 — Screenshot Evidence

Take screenshots of:

- billing alert
- S3 Block Public Access page
- IAM policy validation concepts
- CloudTrail dashboard, if enabled

Do not screenshot real account IDs, keys, emails, or sensitive data.
